# 数据结构 (Data Structures) 学习指南

本模块旨在深入浅出地讲解数据结构的核心概念、不同类型的特点、实现方式以及在各种问题中的应用。通过本指南，您将构建扎实的数据结构基础，为算法学习和程序设计打下坚实的基础。

## 学习模块

本模块包含以下五个核心学习单元，建议按照顺序逐步深入：

1.  **数据结构概述与基本操作 (`ds_overview_basics.md`)**
    * **内容提要：** 介绍数据结构的基本概念、逻辑结构（如集合、线性、树、图）与物理存储结构（如顺序、链式）的区别。详细讲解数据结构的基本操作：插入、删除、查找、遍历等。
    * **整合自：** 原 `A3_DataStructure/data_structure/` 下的 `3. 数据结构.ipynb`, `3.1 存储结构.ipynb`, `3.2 逻辑结构.ipynb`, `3.3 基本操作.ipynb`。

2.  **线性数据结构 (`linear_data_structures.md`)**
    * **内容提要：** 深入学习数组、链表、栈和队列这四种最基本和最常用的线性数据结构。涵盖它们的特点、基于不同存储结构的实现方式、各种操作的复杂度分析以及在实际问题中的应用。
    * **整合自：** 原 `A3_DataStructure/data_structure/` 下的 `4. 数组与链表.ipynb`, `5. 栈与队列.md`。

3.  **哈希表 (`hash_tables.md`)**
    * **内容提要：** 专注于哈希表这一高效查找数据结构的原理。讲解哈希函数的设计、哈希冲突的解决机制（如开放地址法、链地址法），以及哈希表在字典、缓存等场景中的应用。
    * **整合自：** 原 `A3_DataStructure/data_structure/` 下的 `6. 哈希表.md`。

4.  **树结构 (`tree_structures.md`)**
    * **内容提要：** 深入学习树的基本概念、各种树的表示方法、二叉树的特性及遍历（前序、中序、后序、层序）。包含平衡二叉树（如AVL树、红黑树）和多路查找树（如B树、B+树）等高级树结构。
    * **整合自：** 原 `A3_DataStructure/data_structure/` 下的 `7. 树.ipynb`, `7.1 树的遍历.ipynb`, `7.2 B+树.md`。

5.  **堆与图 (`heaps_graphs.md`)**
    * **内容提要：** 讲解堆（如最大堆、最小堆）的特性、实现以及在优先队列、堆排序中的应用。同时，深入学习图的表示方法（邻接矩阵、邻接表）、图的遍历（DFS、BFS）以及图在最短路径、最小生成树等算法中的应用。
    * **整合自：** 原 `A3_DataStructure/data_structure/` 下的 `8. 堆.md`, `9. 图.md`。

# 算法 (Algorithms) 学习指南

本模块旨在深入浅出地讲解算法的核心概念、设计范式和分析方法，帮助学习者构建扎实的算法基础，并掌握常见算法的实现与应用。

## 学习模块

本模块包含以下六个核心学习单元，建议按照顺序逐步深入：

1.  **算法概论与分析 (`algorithms_overview.md`)**
    * **内容提要：** 介绍算法的基本定义、设计思路、算法的描述方法以及如何从宏观层面评估算法质量。此模块旨在为算法学习打下理论基础。
    * **相关实践：** 包含对算法初步认识、算法设计过程、算法质量评估、算法应用的 Jupyter Notebooks。

2.  **复杂度分析 (`complexity_analysis.md`)**
    * **内容提要：** 专注于算法的时间复杂度和空间复杂度分析。详细讲解大O表示法，并通过实例分析不同算法的效率特性，是衡量算法优劣的关键。
    * **相关实践：** 包含复杂度分析相关的 Jupyter Notebooks，如时间复杂度、空间复杂度实践。

3.  **查找与排序算法 (`searching_sorting.md`)**
    * **内容提要：** 系统学习各种经典的查找算法（如二分查找、哈希查找）、排序算法（如冒泡排序、快速排序、归并排序、堆排序等）以及关联规则挖掘算法（如Apriori、FP-Growth、ECLAT）。涵盖其原理、实现细节及适用场景。
    * **相关实践：** 包含查找、排序和关联规则挖掘的Jupyter Notebooks，如市场购物篮分析、推荐系统实现等。

4.  **动态规划 (`dynamic_programming.md`)**
    * **内容提要：** 深入探讨动态规划这一重要算法设计范式。讲解最优子结构、重叠子问题、状态转移方程的构建，并通过经典问题（如背包问题、最长公共子序列）掌握其应用。

5.  **分治与回溯 (`divide_conquer_backtracking.md`)**
    * **内容提要：** 学习分治算法（将大问题分解为小问题解决）和回溯算法（通过尝试和撤销来搜索解空间）的核心思想、递归实现和在各种问题中的应用。

6.  **贪心算法 (`greedy_algorithms.md`)**
    * **内容提要：** 讲解贪心算法的原理、贪心选择性质和最优子结构，并通过典型问题（如霍夫曼编码、最小生成树、活动选择问题）理解其应用范围和局限性。

## Jupyter Notebooks

在本目录下，您还会找到与上述模块内容配套的 Jupyter Notebooks，它们提供了代码实现、可视化和交互式练习，以帮助您更好地理解和实践算法。这些 Notebooks 与 `.md` 文档相辅相成，建议在学习对应理论模块后进行实践。
